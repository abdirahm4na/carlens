# CarLens Product Requirements Document (PRD)

## Overview

CarLens is an AI-powered mobile application that allows users to identify vehicles from photos and instantly receive useful information about them.

The first version focuses on creating a simple, reliable experience that helps users understand a car before buying or learning more about it.

---

# Problem

People often find a car they like but have to search multiple websites to learn:

- What car is this?
- Is it reliable?
- What problems does it have?
- Is it worth buying?
- What is it worth?

CarLens answers those questions in one place.

---

# Goal

Allow a user to:

1. Take a picture of any vehicle.
2. Identify the vehicle.
3. View important information.
4. Ask AI questions about that vehicle.

The experience should take less than 30 seconds.

# Version 1 Features

## Authentication

- Sign Up
- Login
- Logout

---

## Scan Vehicle

User can:

- Open camera
- Upload image
- Scan vehicle

---

## AI Identification

Display:

- Make
- Model
- Year range
- Confidence score

---

## Vehicle Information

Display:

- Engine
- Horsepower
- Drivetrain
- Fuel economy
- MSRP
- Body style

---

## Reliability

Display:

- Reliability score
- Common issues
- Average maintenance cost

---

## AI Assistant

Allow users to ask:

- Is this reliable?
- Is this worth buying?
- Is 120k miles too much?
- What should I inspect?

---

## Scan History

Users can:

- View previous scans
- Open previous results

# Success Metrics

A successful scan should:

- Finish within 15 seconds
- Identify the correct vehicle
- Provide useful information
- Encourage the user to ask questions
