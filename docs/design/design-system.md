# CarLens Design System

## Brand Personality

CarLens should feel:

- Modern
- Intelligent
- Trustworthy
- Fast
- Minimal

---

# Colors

## Primary

Blue

Hex: #2563EB

Used for:
- Buttons
- Links
- Active states

---

## Success

Green

Hex: #22C55E

Used for:
- Reliability score
- Success messages

---

## Warning

Orange

Hex: #F59E0B

Used for:
- Warnings
- Maintenance alerts

---

## Danger

Red

Hex: #EF4444

Used for:
- Severe issues
- Errors

---

## Background

Light Gray

Hex: #F8FAFC

# Icons

Use Lucide Icons.

Examples:

- Camera
- Car
- Search
- Heart
- Bookmark
- User
- Settings

---

# Animations

Keep animations under 300ms.

Animations should feel smooth but subtle.

Examples:

- Fade
- Slide
- Scale
