# 🚗 CarLens

An AI-powered mobile application that identifies vehicles from photos and provides specifications, reliability insights, market values, and AI-powered buying advice.

---

## Features

- 📸 Scan any vehicle
- 🤖 AI vehicle identification
- 📊 Specifications
- ⭐ Reliability score
- 💰 Market value
- 🔧 Common issues
- 💬 AI assistant
- 📚 Scan history

---

## Tech Stack

- React Native
- Expo
- TypeScript
- Supabase
- OpenAI
